Changelog

Brug denne chagngelog til ændringer, opdateringer, justeringer. En subsection per version, dette skal med i hver release.

  • Angiv release date
  • Lav grupperinger under disse kategorier.
  • Added for nye features.
  • Changed for ændringer i eksisterende funktionalitet.
  • Deprecated for features der bliver fjernet i fremtidlige releases.
  • Removed for deprecated features der er fjernet fra denne release.
  • Fixed bug fixes.
  • Security to invite users to upgrade in case of vulnerabilities.
